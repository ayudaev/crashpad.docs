Wraps the gtest ASSERT_DEATH() macro to make assertions about death caused by CHECK() failures.

In an in-Chromium build in the official configuration in Release mode, CHECK() does not print its condition or streamed messages. In that case, this macro uses an empty regex pattern when calling ASSERT_DEATH() to avoid looking for any particular output on the standard error stream. In other build configurations, the regex pattern is left intact.
