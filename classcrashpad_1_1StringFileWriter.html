Detailed Description

A file writer backed by a virtual file, as opposed to a file on disk or other operating system file descriptor-based file.

The virtual file is a buffer in memory. This class is convenient for use with other code that normally expects to write files, when it is impractical or inconvenient to write a file. It is expected that tests, in particular, will benefit from using this class.
